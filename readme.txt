Add A Copyright Footer With Current Year And Sitename

== Description ==

Add a copyright widget to the website footer. The widget will display the current year and the sitename. Can add a start year to show as "Copyright 2010 - 2014 Smiths Grocery" 

Also the ability to add plain text or HTML before and after the copyright. If your system supports shortcodes in widgets, you can add shortcodes to display anything.

== Installation ==

This section describes how to install and use the plugin.

1. Install normally using the WordPress Plugin installer
2. Activate the plugin
3. Add the widget to the footer sidebar
